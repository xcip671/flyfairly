## 怎么用Fly Fairly订票?

流程其实很简单:

1. **搜索航班** - 输入出发地、目的地和日期
2. **选择航班** - 从650+航空公司里挑合适的
3. **选择支付方式** - BNPL、加密货币、信用卡随你挑
4. **确认订单** - 收到确认邮件和航空公司的订单号(PNR)

如果选了分期付款,比如用Klarna或Afterpay:
- 下单时付第一笔(通常是总额的25%)
- 剩下的分3-4次,每两周自动扣款
- 只要按时还,就是0利息

用加密货币的话就更简单,选择你想用的币种,完成支付,立刻收到确认。
